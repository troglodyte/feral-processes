# petmud

An ASCII game blending Pokemon (creature taming/battling), Palworld (tamed
creatures work for you at a base), and Dwarf Fortress (systemic simulation,
procedural worlds, configurable permadeath).

Single-player first, built in Rust as a terminal (TUI) app, with the
simulation kept decoupled from rendering so a client/server split is
possible later.

See the implementation plan for architecture and MVP scope.
